# licensius
Collection of RDF licensing projects

See: 
- https://github.com/oeg-upm/licensius/odrlapi Evaluator for the newest ODRL spec
- https://github.com/oeg-upm/licensius/rdflicense Collection of common licenses as RDF
- https://github.com/oeg-upm/licensius/jodrlapi Java library to manipulate ODRL 2.1 expressions
- https://github.com/oeg-upm/licensius/licensius-core Java library to find common licenses in HTML web pages
- https://github.com/oeg-upm/licensius/licensius-ws Java HTTP Rest API to find common licenses in HTML web pages
